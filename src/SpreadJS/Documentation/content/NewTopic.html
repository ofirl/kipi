    You can copy styles and images from Excel and paste to worksheets in SpreadJS. This makes it easy for you to copy Excel styles and images without having to import the entire spreadsheet. You can enable copying and pasting of styles and images by setting the allowCopyPasteExcelStyle property to true. For example (spread.options.allowCopyPasteExcelStyle = true;).

    Excel Styles

    Set the allowCopyPasteExcelStyle property to true to allow copying and pasting of styles.

    Floating Objects

    You can also copy images from Excel and paste them into your worksheets in SpreadJS by setting the allowCopyPasteExcelStyle property to true. Further, you can also copy charts and shapes from Excel and paste them in SpreadJS as images. However, this feature is already enabled in SpreadJS by default, but it does not provide support for copying and pasting the images from SpreadJS to Excel.

    You can use the Ctrl + V and Command + V keyboard combination in Windows and Mac while pasting the images in SpreadJS worksheets as shown in the following image.

    You can observe that there was no impact on the size and the content of the original image while pasting it in SpreadJS. However, you can resize the pasted image by using the resize handlers. The location of the image in worksheet is determined by the active cell of SpreadJS. You can also paste multiple images in SpreadJS, but it is always recommended to paste them one by one in SpreadJS worksheet, or else it will copy all the images in a single big image.
